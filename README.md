# WHEAT
为微前端架构管理前端项目
## get start
```bash
# clone from git
$ git clone https://github.com/node-wheat/wheat.git

# install dependencies
$ yarn

# start server with development mode
$ npm run server

# start server with development mode
$ npm run server
```
